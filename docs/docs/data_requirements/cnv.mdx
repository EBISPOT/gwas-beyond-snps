# CNV data requirements

This page describes the columns that should be present in your Copy Number
Variant (CNV) summary statistics file for submission to the GWAS Catalog.

Column names must appear exactly as shown below.
Any differences or typos in your file will cause validation errors.

Files are expected to contain at least **100,000 rows**. Smaller files may be
rejected or flagged for review.
