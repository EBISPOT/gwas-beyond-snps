Welcome to the GWAS Catalog summary statistics tool 🧬🧪

This tool will check your summary statistics files are valid and automatically create new files ready for submission.

Your data stays on your computer — nothing is uploaded to a server. All validation runs locally in your browser using Pyodide (Python compiled to WebAssembly).

If you need help, email gwas-subs@ebi.ac.uk.

Thank you for submitting your data to the GWAS Catalog ❤️

What kind of genetic variation have you studied?

Why do we ask this?

Different types of genetic variants have different data requirements. For example, structural variants like CNVs require start and end positions (a range), while SNPs only require a single position.

Select variation type

SNP submissions are not supported by this tool.

Please see gwas-sumstats-tools for SNP submissions.

Only SNP-based, CNV-based, and gene-based genome-wide studies are currently validated by the GWAS Catalog.

To enquire about submitting a non-standard data type, please contact gwas-subs@ebi.ac.uk.

How have you measured the effect size of your variants?

Why do we ask this?

Effect size and uncertainty measurements are mandatory for most GWAS submissions. Knowing which measure you used helps us validate your data correctly.

Select effect size measure

All CNV and SNP submissions to the GWAS Catalog must include an effect size estimate. Please contact gwas-subs@ebi.ac.uk if you have questions.

However, effect sizes are optional (but encouraged) for gene-based GWAS.

How are p values represented in your data?

Why do we ask this?

The GWAS Catalog accepts standard p values or negative log₁₀ p values (not both).

Select p-value representation

Are highly significant p values thresholded or rounded to zero?

Why do we ask this?

Thresholding or rounding p values limits the downstream usability of the data. If your data includes p values thresholded or rounded to zero, please provide negative log₁₀ p values instead.

Are p-values thresholded?

If your data includes p values thresholded or rounded to zero, please provide negative log₁₀ p values instead.

If this isn't possible, you will be asked to include more details about the GWAS software that you used in the study metadata.
